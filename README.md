## 项目结构
- **core**：核心模块，包含项目的主要功能和逻辑。
    |-- **engine.py**：引擎模块，负责处理项目的核心算法和逻辑。
    |-- **anomaly_wrapper.py**：异常处理模块，负责捕获和处理项目中的异常情况。
    |-- **musc_wrapper.py**：MUSC模块，负责与MUSC系统进行交互和数据处理。
    |-- **data_bridge.py**：数据桥模块，负责连接和转换不同数据源的数据。

- **app**：应用模块，包含项目的用户界面和交互逻辑。
    |-- **main_app.py**：主应用模块，负责启动和管理整个应用程序。

- **cloud**：云模块，包含与云服务相关的功能和逻辑。

## 安装和使用
1. 克隆项目到本地：
   ```bash
   git clone <repository-url>
   ```
2. 安装依赖：
    ```bash
    pip install -r requirements.txt
    ```
3. 运行应用程序：
    ```bash
    python app/main_app.py
    ```
